Location Finder plugin is designed for Auto Classifieds and Realty Classified Software nevertheless it can be used for any Flynax classifieds script. It can accurately detect the location of your apartment, house or any other vehicle or a boat on maps.

This plugin creates a new group for the pages "Add/Edit Listing" with fields where users can type their addresses and make a search for a needed listing of an apartment or a car. The information is adequate because the plugin uses longitude and latitude.

This plugin uses Google Maps that's why the information is always accurate and up-to-date.
