Flynax Listing Export/Import plugin can help classifieds website owners to easily import and export listings data from XLS (Excel) files and CSV files (with comma separator).

Listing Export/Import plugin offers the most flexible way of listings transfer from and to classifieds websites. The flexibility of it is that you may open a file with the help of Excel and edit, add or remove pics, owner change of listings at any stage of the import/export process.

The other and important feature of the plugin is that it offers import of pictures using an archive with pics or through pics URLs of imported files. So you get fully functional listings with pic galleries after completion of import.

One more useful thing that this plugin can help you to transfer listings between Flynax powered classifieds websites or make backup of listings.
