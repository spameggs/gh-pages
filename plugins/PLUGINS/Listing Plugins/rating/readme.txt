Listing Rating plugin adds a new section to pages with listing details. This section contains a bar with stars that can be clicked by users and set ratings for a listing. The bar with star has a short statistics on ratings of a listing that displays the number of votes and the average rate.

The plugin also adds a rating with small stars in grid of listings on pages with search results or when users browse those pages.

By installing an additional Flynax plugin like Listings Box with Listing Rating plugin you get an option to create blocks with top rated listings.

Listing Rating plugin has several settings that allow you to change the number of stars and prohibit guest visitors to rate listings.
