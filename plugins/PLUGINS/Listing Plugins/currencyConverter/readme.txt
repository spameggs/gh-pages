Flynax offers a free Currency Converter plugin for its Classifieds Scripts.
This plugin allows users to count any price for a car or any other classified in their own currency. There is no need to count by yourself because the Currency Converter plugin will do it for you. As for the correct exchange rates than the plugin updates them on a daily basis so users will get the "fresh".

The currency is selected depending on the location of the visitor and plugin converts listings’ prices using that auto preference which makes it easy to use.
