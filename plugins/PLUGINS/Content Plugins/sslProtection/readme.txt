Flynax SSL Protection Plugin is a simple but useful plugin which transfers particular pages from http to https protocol. This plugin adds a new option for add/edit pages of the administration interface. Using this option an administrator may choose particular page that will be accessible through https protocol.

Flynax SSL Protection Plugin also has an option that enables https protocol for all pages of the customer area.

Important notice: Before installation of the plugin, please be sure that you purchased and installed SSL certificate for your classifieds website, otherwise this plugin will not work.

What is SSL Certificate? Read more here. http://tldp.org/HOWTO/SSL-Certificates-HOWTO/x64.html
