Flynax Data Entries Import plugin lets you easily manage any types of things you want to be available on a page with listings.

For example if you want to make all types of bicycles available in the drop down menu on a page with a listing you just need to find the types of bicycles in a particular file extension such as xls, csv or txt and use the Data Entries Import plugin.
Using this plugin you define the location of the file with types of bicycles by making a few clicks then you get all types of bicycles in your software field.
