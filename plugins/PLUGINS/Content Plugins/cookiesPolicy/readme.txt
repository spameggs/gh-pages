Cookies Policy plugin prevents use of your classifieds website by users without their agreement for using browser cookies.

If a user agrees with the Cookies policy the website will save user’s choice and allows using the website. If a user rejects this policy then the website redirects him to another website which set in the settings of the plugin by an administrator.

This feature is needed only for those countries where is introduced a law on use of users' cookies mainly in European countries. The plugin is activated for European countries by default, but an administrator may change the list with countries in the plugin's settings.

In the folder of the plugin you will find a PSD file with the graphic source that is used by the plugin and may amend the color from yellow to any other you want.
