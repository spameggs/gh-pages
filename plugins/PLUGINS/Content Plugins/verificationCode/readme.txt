Flynax Verification Code Plugin is the simplest but useful plugin that can be used for classifieds websites. With the help of the Flynax Verification Code plugin you may add META tags into the header or footer of your classifieds website for activation or verification of any web services like Google Webmaster Tools or any other.

This plugin also allows adding Javascript code for activation of different trackers like Alexa, statistics services like Google Analytics or chat codes.

META tags can be placed on all your website's pages or can be used for particular ones, where you want to activate or use a service. By placing a code you may choose a position for it on your page. In this case you may use META tags for header and footer for Javascript.
