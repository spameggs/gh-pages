Flynax Category Filter Plugin lets you arrange blocks with filters by different fields for particular categories or for all categories of any listing type.

Category Filter Plugin provides you with this useful feature that becomes a convenient tool for users as they use filters for categories with a lot of listings. With this plugin your site users can filter listings by particular criteria and sort out only necessary classifieds.

Category Filter Plugin offers one more option that allows using slides and groups ranges for such kind of fields as numeric or price and makes the types more convenient for users. The data of each filter is added into META title of a page that makes the content unique for every filter link.
